# U-bent-optical-fiber-sensor

This is a small Arduino project based on a simple heart rate sensor.

## Build setup

* Hardware:
	* Arduino UNO (or any other)
	* Heart rate sensor
  * OLED display (optional)

## Wiring

### Heart rate sensor (HRS):

HRS pin | Arduino pin
--- | ---
\+ | +5V
\- | GND
S | A1

### OLED display (optional):

OLED pin | Arduino pin
--- | ---
VDD | +5V
GND | GND
SDA | A4
SCK | A5
